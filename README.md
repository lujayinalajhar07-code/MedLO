# 🩺 MedLuj — The Ultimate Medical Study Universe

مصنوع بكل المحبة من لجين لـ عبادة 💛

## هيكل المشروع

```
medluj/
├── app.py                    ← Flask backend (Login + Routes)
├── requirements.txt
├── templates/
│   ├── login.html            ← صفحة تسجيل الدخول الزجاجية
│   └── index.html            ← لوحة التحكم الرئيسية الكاملة
└── static/
    └── data/
        └── medluj_data.json  ← قاعدة البيانات المحلية (رسائل + طبي)
```

## تشغيل المشروع

```bash
pip install -r requirements.txt
python app.py
```

ثم افتح: http://localhost:5000

## المميزات

### 🕐 قسم التايمر
- عداد تصاعدي بستايل ECG نيوني
- نبضات القلب تُملأ كل ساعة دراسة
- رسائل تشجيعية عند 30 دقيقة، ساعة، ساعتين...
- تايمر Pomodoro (25 دقيقة عمل / 5 دقائق راحة)
- حفظ تلقائي بالـ localStorage

### 📊 قسم الإحصائيات
- رسم بياني أسبوعي بـ Chart.js
- دائرة إنجاز نيونية (من أصل 50 ساعة أسبوعياً)
- إحصائيات: أفضل يوم، الأيام المتواصلة، مجموع الأسبوع

### 💌 رسائل لجين
- 15 رسالة تشجيعية مبرمجة
- عرض عشوائي
- تصنيف بحسب الحالة المزاجية

### 🔬 الأدوات الطبية
- بحث في قاموس المصطلحات (20 مصطلح)
- فاحص القيم الحيوية (7 قياسات) مع تنبيهات طبيعي/مرتفع/منخفض
